This repository contains notes and summaries related to core electronics concepts used in embedded and automotive systems.

## Topics Covered

* Diodes
* BJT
* MOSFET
* Rectifiers
* Voltage Regulators
* Op-Amps
* Comparators
* Digital Logic

## Purpose

The goal of this repository is to strengthen electronics fundamentals required for embedded hardware and automotive electronics understanding.

## Contents

Each topic may contain:

* Concept summaries
* Basic circuit explanations
* Device applications
* Interview-oriented notes
